# Sekretariat HMTI

> WebGL | ThreeJS

[![NPM Version][npm-image]][npm-url]

Sekretariat HMTI hadir dalam bentuk 3D melalui platform website dibuat dengan ThreeJS yang menggunakan engine WebGL

![](header.png)

## Installation

NPM :

```sh
npm install
```

Yarn :

```sh
yarn install
```
